# 🎯 Quant Pivot Engine

🚀 A comprehensive pivot-based trading system implementing **Fibonacci, CPR, and Camarilla pivot levels** to identify key market turning points.

---

## 🧠 Overview

This system computes multiple pivot frameworks used in professional trading:

* 📊 **Fibonacci Pivots** — for trend continuation and reversal zones
* 📉 **CPR (Central Pivot Range)** — for market structure and volatility analysis
* ⚡ **Camarilla Levels** — for intraday mean reversion strategies

It helps traders identify **high-probability support and resistance zones**.

* Based on Fibonacci ratios
